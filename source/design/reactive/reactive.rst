Реактивное программирование
############################

.. container:: left-col

    Реактивное программирование — это программирование с асинхронными потоками(streams) данных.

    В общем говоря, здесь нету ничего нового. Шины событий или типичные события нажатий на кнопки — это все реальные примеры асинхронных событийных потоков, которые вы можете слушать и выполнять некоторые побочные действия

    Реактивное программирование — событийно-ориентированое программирование, событие на входе, события на выходе.

The Reactive Manifesto
**********************

.. container:: left-col

    Responsive
        The system responds in a timely manner if at all possible

    Resilient
        The system stays responsive in the face of failure

    Elastic
        The system stays responsive under varying workload (при изменяющейся нагрузке)

    Message Driven
        Reactive Systems rely on asynchronous message-passing to establish a boundary between components that ensures loose coupling, isolation and location transparency
