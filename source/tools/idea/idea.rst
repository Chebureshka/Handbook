Навигация
=================
.. container:: left-col

    .. container:: code-markup

        Alt+F1
            Перейти в навигационную панель по структуре проекта

        Alt+Home 
            Перейти к NavigationBar, относительно текущего файла

    .. rubric:: Недавние файлы
    
    .. container:: code-markup

        Ctrl+E
            Показать недавно открываемые файлы. Еще раз **Ctrl+E** - показать только измененные файлы.

        Ctrl+Tab
            Переключение между текущими открытыми открытыми. Удерживать **Ctrl+Tab** - открывает окно со всеми открытыми файлами.

        Ctrl+Shift+E
            Недавно открываемые/измененные места в файлах.

    .. rubric:: Поиск

    .. container:: code-markup

        Ctrl+Shift+F
            Глобальный поиск по символам

        Ctrl+N
            Перейти в поиск по имени класса

        Ctrl+Shift+N
            Перейти в поиск по имени файла

        Alt+Ctrl+Shift+N
            Перейти в поиск по символу в коде

    .. rubric:: Маркеры

    .. container:: code-markup

        F11 
            Поставить/Снять маркер

        Ctrl+F11
            Поставить именованный маркер

        Ctrl+{1-0}
            Перейти к маркеру под указанным номером

        Shift+F11
            Перейти к общему списку маркеров

    .. rubric:: Навигация внутри класса

    .. container:: code-markup

        Ctrl+F12
            Посмотреть все методы в текущем классе

        Alt+Стрелка
            Навигация по методам

Набор кода и рефакторинг
===========================
.. container:: left-col

    .. container:: code-markup

        Ctrl+J
            Посмотреть и выбрать live-template для вставки

        Ctrl+Space 
            Auto-complite названия из контекста

        Ctrl+Shift+Space 
            Smart Auto-complite названия из контекста

        Ctrl+Shift+Enter
            Statement Auto-complite

        Ctrl+Shift+Стрелка
            Переместить строку на которую указывается курсор, или все выделенные строки
